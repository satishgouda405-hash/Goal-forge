# GoalForge 🎯

A goal-tracking app — offline-first, works in any browser.

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Click **Deploy** — no settings needed!

## Deploy on GitHub Pages
1. Push to GitHub
2. Go to repo **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your app will be live at `https://yourusername.github.io/goalforge`

## Files
- `index.html` / `goalforge.html` — the full app (same file)
- `vercel.json` — Vercel config
